Overview of the functionalities of Splash

These are the basic functionalities:

  1. Start page - first in order to have access to the application you need to login (user:admin password:admin).

  2. Search - after providing credentials you can search for the customers. On the top navigation you can see the user name and you can change your language from German to English.

  3. Dashboard - to access all the functionalities you can check the section “Dashboard” to have a glance at all the possibilities with splash. Here you can search for the customers, create a new one by providing the essential data and also check and update your data under the “Personal Data” section.

  4. New Customer - to create a new user, you need to ptovide the information, then you can add a new customer to the list.

  5. Afterwards you can see that the new customer is added to the list of customers and by clicking on it you can update the information.If you know the password (it's in the database), you can log in as a new user.

  6. Personal Data - here you can create a new password for the customer only if you are an admin user. The role can only be changed in the database.
